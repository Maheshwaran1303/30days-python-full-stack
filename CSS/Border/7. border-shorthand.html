CSS Border - Shorthand Property

The border property is a shorthand property for the following individual border properties:

  • border-width
  • border-style (required)
  • border-color

This property is a shorthand property for border-width (5px), border-style(solid), and border-color(blue).


You can also specify all the individual border properties for just one side:

The border-left Property

This property is a shorthand property for border-bottom-width, border-left-style, and border-left-color.
