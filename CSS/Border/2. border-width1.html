CSS border-width Property

This property specifies the width of the four borders:

The width can be set as a specific size (in px, pt, cm, em, etc..) or by using one of the three pre-defined values: thin, medium, or thick:

border-style: solid; border-width: thin

border-style: solid; border-width: medium

border-style: solid; border-width: thick

border-style: solid; border-width: 10px

border-style: dotted; border-width: thin

border-style: dotted; border-width: medium

border-style: dotted; border-width: thick

border-style: dotted; border-width: 10px

border-style: dashed; border-width: thin

border-style: dashed; border-width: medium

border-style: dashed; border-width: thick

border-style: dashed; border-width: 10px

border-style: double; border-width: thin

border-style: double; border-width: medium

border-style: double; border-width: thick

border-style: double; border-width: 10px

Note: The "border-width" property does not work if it is used alone. Always specify the "border-style" property to set the borders first.
